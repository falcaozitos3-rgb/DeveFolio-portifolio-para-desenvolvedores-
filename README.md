# 📂 Devfolio

O **Devfolio** é um sistema web de portfólio dinâmico desenvolvido para gerenciar, armazenar e expor projetos e habilidades de forma automatizada através do Back-end.

## 💻 Tecnologias e Engenharia do Sistema

*   **Ambiente de Servidor:** Desenvolvido em Python utilizando o micro-framework **Flask** para o roteamento e controle da lógica de negócios.
*   **Camada de Persistência:** Integração com **SQLite** utilizando **SQLAlchemy** para criar um banco de dados leve, rápido e embutido, ideal para persistência de dados estruturados sem sobrecarregar o hardware.
*   **Interface e UI/UX:** Estruturação base em HTML5, CSS3 e JavaScript. O design visual e a responsividade foram refinados com o suporte de ferramentas de Inteligência Artificial, permitindo uma interface polida enquanto o foco do desenvolvimento permaneceu concentrado na arquitetura do Back-end.

## 🧠 Funcionalidades Implementadas no Back-end

O coração do sistema foi construído focando em boas práticas de desenvolvimento Back-end:
*   **Manipulação de Dados:** Operações de CRUD integradas ao SQLite para gerenciar as informações exibidas nas telas.
*   **Controle de Rotas:** Roteamento dinâmico no Flask para renderização segura dos templates.
*   **Segurança Básica:** Validação e tratamento de dados recebidos via requisições no servidor.

## 🚀 Como Executar Localmente

1. Certifique-se de ter o Python instalado na máquina.
2. Instale as dependências listadas no projeto:
   ```bash
   pip install -r requirements.txt
   ```
3. Execute a aplicação:
   ```bash
   python app.py
   ```
4. Acesse o endereço local indicado no terminal (padrão: `http://127.0.0.1:5000`).
